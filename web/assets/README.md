# ZubiLook website media

These files are the production media used by `web/index.html`.

- Images and videos are AI-generated product demonstrations, not clinical results.
- They are stored locally so the website does not depend on expiring generator CDN URLs.
- `demo-flow.mp4` and `story-transformation.mp4` are intentional exceptions to the
  repository's 1 MB binary guideline. Both remain below GitHub's per-file limit.
- Keep the visible AI-visualization disclaimers in the website whenever these assets
  are used.
